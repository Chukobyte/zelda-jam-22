## Zelda Jam 22 Game

A work in progress game being made for the [KB Games Zelda Jam '22](https://itch.io/jam/zeldajam-22).  Made with [Seika Engine](https://github.com/Chukobyte/seika-engine).

### Controls

#### Keyboard

- Move: WASD, Arrow Keys
- Attack: Space
- Debug Exit: ESC

#### Controller (Xbox)

- Move: Left Analog
- Attack: A Button
- Debug Exit: Back Button
